Job Cost Sheet Integration with Material Planning of Job Order

Product Material Planning Integration with Job Cost Sheet on Job Order / Work Order - Odoo App

This app allows you to create and edit the Job Cost Sheet from Material Plannings from the Job Order / Work Order.


Main Features:

  • Allow Project Manager to create a Job Cost Sheet using Material Plannings from the Job Order / Work Order form.
  • Allow Project Manager to Add Material Plannings on Job Order and Edit Job Cost Sheet using Material Planning from Job Order.
  • Then the Project Manager can see the Job Order Reference / Link on the Job Cost Sheet.
  • For more details please see below screenshots and watch the video.
