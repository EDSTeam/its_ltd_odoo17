Expense Submit for Project and Job Order / Work Order in Odoo

This app allows your project users who are working on a Job order or Work orders for construction and job contracting projects in the system to submit their expenses with pictures of expenses to the expense manager and expense team for reimbursement.


Main Features:

  • Allow your employee / user to submit expenses for the job order / work order they are working on.
  • On Expense form a user can select project, job order and photos for expense as shown.
  • User/Employee Submit Expenses to Expense Manager. (Odoo Standard Flow) for reimbursement.
  • Project user(s) can see the Expenses and/or Expense Sheets using the smart button on the Job Order / Work Order / Task form.
  • Project managers can see the Expenses and/or Expense Sheets from the smart button on the project form.
  • Related smart buttons on Job order and Construction Project to have a quick view of expenses.
  • For more details please see video in the live preview and check below screenshots.
